  CWpcSimulator

・説明
CardWirthのPCシミュレータ
選択された性別、年代・素質・特性からPCの能力・適性を計算し、表示します
要.NET Framework 4.0


・使い方
能力の名称をクリックするとその能力が上昇する年代・素質・特性の文字は青く、下降する物は赤く表示されます
リセットボタンを押すと性別・年代・素質・特性がリセットされます
